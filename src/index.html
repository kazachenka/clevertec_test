Задание для Clevertec

Тестовым заданием для Clevertec я сделал приложение, в котором хранятся данные о фотографах, их альбомах и фото. У каждого фотографа есть по 10 альбомов с уникальными 'title', а у каждого из альбомов по 50 фотографий.

Вся информация для заполнения получена с: https://jsonplaceholder.typicode.com/
